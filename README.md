# CareBot – AI Health Assistant

CareBot is a simple full-stack project that provides basic health suggestions using AI.  
Users can enter their symptoms and get helpful advice generated using a local AI model.

The goal of this project is to understand how AI can be integrated with a web application and how responses can be improved using custom datasets.

---

## Features

- User authentication (Register / Login)
- Chat-based interface
- AI-powered responses using Ollama
- Dataset-based response improvement (basic RAG)
- Multiple symptom detection
- Clean and simple UI

---

## Tech Stack

- HTML

Backend:

- Ollama (Mistral / Phi3)
